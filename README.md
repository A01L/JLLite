# JetLine v8.0 Lite — облегчённое ядро для приложений на PHP
<br><br>
JetLine v8.0 Lite — это минималистичная версия фреймворка JetLine, созданная для быстрого развёртывания веб-приложений без избыточных модулей и сложной маршрутизации.
Лёгкий и оптимизированный для высокой нагрузки - фреймворк ориентирован на разработчиков, которым нужен чистый каркас без лишних зависимостей.
<br><br>
## Ключевые особенности
Маршрутизация <br>
В отличие от полной версии, JetLine Lite не требует ручного распределения трафика по контенерам.
Все входящие запросы автоматически направляются внутрь containers (положил код и запустил на JL Lite).
<br><br>
## Облегчённая архитектура контейнеров <br>

<ul>
<li>Минимальное ядро</li>
<li>Только необходимые модули</li>
<li>Лёгкая система загрузки компонентов</li>
<li>Урезанные тяжёлые сервисы, не влияющие на базовую работу приложений</li>
</ul>

<br>
Lite-версия сохраняет мощь структуры JetLine, но устраняет всё, что не относится к основному пайплайну обработки HTTP-запросов.
<br><br>
## Высокая производительность <br>
Благодаря отсутствию тяжёлых вспомогательных модулей, фреймворк стал легче, быстрее и экономичнее при работе под нагрузкой.
